# Credit Card Fraud Detection

**Intern ID:** CITS4523

## Description
A machine learning model that detects fraudulent credit card transactions
using Random Forest Classifier. The model analyzes transaction patterns
like amount, time, distance, and frequency to identify fraud.

## Libraries Used
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## How It Works
1. Dataset created with 1000 transactions (950 normal, 50 fraud)
2. Features: amount, time of transaction, distance, frequency
3. Random Forest Classifier trained to detect fraud patterns
4. Model predicts whether a new transaction is Fraud or Normal

## Model Performance
- Algorithm  : Random Forest Classifier
- Trees Used : 100
- Test Size  : 20%
